# Studies of Nest

## Description

Some studies about Nest JS

### Tecnologies/Stack

- Typescript
- NestJS
- GraphQL
- Postgres
- Docker
- TypeORM

## Running the app

```bash
$ docker-compose up -d
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
